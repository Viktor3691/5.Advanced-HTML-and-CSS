1. Visual Studio Code

(VS Code) — кроссплатформенный текстовый редактор для разработчиков, разработанный компанией Microsoft. Предназначен для работы с различными языками программирования, а также для разработки кросс-платформенных десктопных приложений.

<<<


2. Atom

(Atom) — кроссплатформенный текстовый редактор, разработанный GitHub. Он основан на библиотеке Node.js и использует JavaScript для реализации пользовательского интерфейса.

<<<


3. Sublime Text

Sublime Text — кроссплатформенный текстовый редактор для написания программного кода на различных языках программирования, а также вёрстки веб-документов. Разработан в 2007–2008 годах австралийским программистом Джоном Скиннером и основанной им компанией Sublime HQ.

<<<


4. Brackets

Brackets — текстовый редактор с открытым исходным кодом, ориентированный на веб-разработку. Создан компанией Adobe, поддерживает HTML, CSS и JavaScript.

<<<


5. Notepad++

Notepad++ (Notepad Plus Plus) — бесплатный текстовый редактор с открытым исходным кодом для Windows. Создан французским программистом Доном Хо в 2003 году.

<<<


6. Kate

Kate (KDE Advanced Text Editor) — текстовый редактор для работы с исходным кодом, разработанный сообществом KDE. Предназначен для разработчиков программного обеспечения.

<<<


7. Vim

Vim — специализированный текстовый редактор для программистов, предназначенный для работы с большими объёмами кода без использования мыши. Создан на основе более старого vi, разработанного Биллом Джойем.

<<<


8. Emacs

Emacs — многофункциональный расширяемый текстовый редактор, который подходит для работы с кодом. В основе редактора — интерпретатор языка Emacs Lisp (Elisp).

<<<


9. Komodo

Komodo Edit — бесплатный текстовый редактор для динамических языков программирования, изданный компанией-разработчиком ActiveState. Был представлен в январе 2007 года в дополнение к коммерческой среде разработки Komodo IDE.

<<<


10. Code Pen

CodePen — онлайн-сообщество для тестирования и демонстрации HTML, CSS и JavaScript-кода, созданного пользователями. Сервис выступает как онлайн-редактор кода и открытая образовательная среда, где разработчики могут создавать и тестировать фрагменты программ (так называемые «Pens»).

11. PhpStorm

PhpStorm — коммерческая кросс-платформенная интегрированная среда разработки (IDE) для PHP, разработанная компанией JetBrains на основе платформы IntelliJ IDEA.

<<<


12. WebStorm

WebStorm — интегрированная среда разработки (IDE) от компании JetBrains, разработанная на основе платформы IntelliJ IDEA. Предназначена для работы с JavaScript, TypeScript, HTML и CSS, а также популярными фреймворками, такими как React, Angular, Vue.js и Node.js.

<<<


13. PyCharm

PyCharm — интегрированная среда разработки (IDE) для языка программирования Python, созданная компанией JetBrains. Название читается как «пайчарм».

<<<


14. IntelliJ IDEA

IntelliJ IDEA — интегрированная среда разработки (IDE) для программирования на различных языках, таких как Java, Kotlin, Groovy, Scala и других. Продукт компании JetBrains.

<<<


15. Visual Studio

Visual Studio — интегрированная среда разработки (IDE), созданная компанией Microsoft. Предназначена для разработки программного обеспечения на различных языках программирования, включая C#, C++, JavaScript, Python, TypeScript, F# и другие.

<<<


16. Net Beans

NetBeans (с 2016 года — Apache NetBeans) — кросс-платформенная интегрированная среда разработки программного обеспечения на различных языках программирования, включая Java, Python, PHP, JavaScript, C, C++, «Ада».

<<<


17. Android Studio

Android Studio — интегрированная среда разработки (IDE) для создания приложений под операционную систему Android, разработанная компанией Google. Основана на программном обеспечении IntelliJ IDEA от компании JetBrains.
