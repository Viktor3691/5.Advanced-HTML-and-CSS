5.5 Расширение VS Code VS HTML to CSS

VS HTML to CSS расширение для VS Code, которое позволяет быстро перенести селекторы из HTML в CSS.

Скриншот

Характеристики

Создаёт шаблон CSS на основе выбранного HTML

  • Обращает внимание только на классы, игнорирует идентификаторы и другие возможные селекторы

    • Настройки расширения

    Открыть Preferences -> Settings

    //Символы, которые будут добавлены после классов CSS (включая пробелы)
    "vs-html-to-css.append": " {\n}\n",
    // Отключить уведомление
    "vs-html-to-css.disableNotification": false,
    // Списки классов, которые CSS должен игнорировать
    "vs-html-to-css.ignore": ["container", "row"],

    Настройка в принципе не нужна. Например, если у вас был следующий HTML

    HTML

    <a class="card" href="#">
    <div class="card_head">
    <img src="#" alt="" class="card_image"></div>
    <div class="card_body">
    <h3 class="card_title">Название</h3>
    <p class="card_text">Текст</p>
    </div>
    </a>

    Выберите весь HTML

    CTRL + a
    , чтобы скопировать следующий CSS в буфер обмена нажмите
    CTRL + c
    перейдите в CSS и нажмите
    CTRL + v

    Получаем

    CSS
    .card {
    }
    .card_head {
    }
    .card_image {
    }
    .card_body {
    }
    .card_title {
    }
    .card_text {
    }

    Если вы хотите добавить дополнительную строку CSS, то просто добавьте его после

    "vs-html-to-css.append": " {\n}\n"

    {\n} ещё один символ \n

    "vs-html-to-css.append": " {\n\n}\n"

    добавляем и получаем CSS код с дополнительной строкой.

    CSS
    .card {

    }
    .card_head {

    }

    }
